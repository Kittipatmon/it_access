ต้อง การ (แบบฟอร์มการร้องขอสิทธิใช้งานเทคโนโลยีสารสนเทศ) แยก frontend backend ให้เป็นระเบียบดูง่าย เเพทเทิร์น model-view-controller หน้าหลัก ให้กรอกแบบฟอร์มการร้องขอสิทธิใช้งาน ข้อมูลที่ต้องกรอกชื่อ,นามสกุล,ชื่อเล่น ,Name,Last Names,Nick Name,โทรศัพท์ โดยจะดึงข้อมูลมากจาก database appkum_user ส่วนหลังบ้านให้ บ้านให้สามารถเห็นบันทึกแบบคำร้อง ให้สิทธิการอนุมัติ แต่ละ step หลักๆจะมี3 step ผู้ร้องขอ, ผู้ตรวจสอบ, ผู้อนุมัติ บันทึกวันที่ ที่อนุมัติ สามารถเพิ่มstepได้ แล้วถ้าอนุมัติครบแล้วให้แจ้งเตือนไปยังผู้ร้องขอ ในหน้าติดตามสถานะ และในส่วนของ หลังบ้านให้ จะ มีจัดการ ผู้ใช้งาน แต่ไม่มีการลบ หรือ แก้ไข ข้อมูล
